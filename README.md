# 아프리카TV Bandal

아프리카TV의 사이드바 UI/UX를 개선하는 크롬 익스텐션입니다.  
홈페이지, 웹 플레이어 좌측에 사이드바를 변경합니다.

## 기능

- 웹 폰트를 `Pretendard`로 변경합니다.
- 즐겨찾기한 BJ의 목록만 사이드바에 보여집니다.
- 시청자 수, 제목, 카테고리를 한눈에 표시합니다.
- 마우스을 올리면 방송 썸네일 이미지를 표시합니다.
